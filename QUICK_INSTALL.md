## üöÄ Installation - Drop into Companion

### Quick Install (Recommended)

1. **Copy the package file to your desktop or downloads folder:**
   ```bash
   cp /Users/trevormarr/Apps/showcall-companion/companion-module-showcall-2.1.0.tgz ~/Desktop/
   ```

2. **In Companion**, there are a few ways to install:

   **Method A: Via File Manager**
   - Open Companion's module directory
   - Extract the `.tgz` file there
   - Restart Companion

   **Method B: Using Terminal**
   ```bash
   # Find your Companion directory (usually ~/companion-satellite)
   cd ~/companion-satellite/bundled-modules/
   
   # Copy and extract
   cp ~/Desktop/companion-module-showcall-2.1.0.tgz .
   tar -xzf companion-module-showcall-2.1.0.tgz
   mv package companion-module-showcall
   
   # Restart Companion
   ```

3. **Add Connection in Companion:**
   - Open Companion UI
   - Go to "Connections" tab
   - Click "+ Add Connection"
   - Search for "ShowCall"
   - Configure:
     - Server: `localhost` (or your ShowCall IP)
     - Port: `3200`
   - Save

4. **Test It:**
   - Start ShowCall v2.3.0
   - Create a preset in ShowCall
   - Watch it appear automatically on your Stream Deck! üéâ
