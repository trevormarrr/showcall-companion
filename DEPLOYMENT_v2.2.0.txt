# Version 2.2.0 Deployment

## ✅ CONFIRMED CHANGES

### Preset Button Names - NOW PULL FROM VARIABLES
✅ **Verified:** Preset buttons now use dynamic variable text:
```javascript
text: index < 8 ? `$(showcall:preset_${index + 1}_name)` : preset.label || preset.id
```

This means:
- Presets 1-8 display: `$(showcall:preset_1_name)` through `$(showcall:preset_8_name)`
- Names are pulled from ShowCall preset data
- Updates automatically when ShowCall preset names change
- Falls back to preset.label or preset.id if variables unavailable

### New Preset Name Variables Added
✅ **8 new variables for preset names:**
- `preset_1_name` - Preset 1 name from ShowCall
- `preset_2_name` - Preset 2 name from ShowCall
- `preset_3_name` - Preset 3 name from ShowCall
- `preset_4_name` - Preset 4 name from ShowCall
- `preset_5_name` - Preset 5 name from ShowCall
- `preset_6_name` - Preset 6 name from ShowCall
- `preset_7_name` - Preset 7 name from ShowCall
- `preset_8_name` - Preset 8 name from ShowCall

### Variable Population Confirmed
✅ **updateVariables() now populates preset names:**
```javascript
const presetVariables = {}
if (this.showcallPresets && Array.isArray(this.showcallPresets)) {
    this.showcallPresets.forEach((preset, index) => {
        if (index < 8) {
            presetVariables[`preset_${index + 1}_name`] = preset.label || preset.id
        }
    })
}
```

## 🎯 KEY FEATURES v2.2.0

### Dynamic Preset Button Names ⭐
Preset buttons now pull their text directly from `$(showcall:preset_N_name)` variables, which are populated from ShowCall preset metadata. This ensures buttons always show the actual preset name defined in ShowCall.

### Real-Time Synchronization
When ShowCall preset names are updated, the companion module:
1. Receives the updated preset data
2. Extracts the preset label/name
3. Updates the `preset_N_name` variable
4. Button text refreshes on Stream Deck
